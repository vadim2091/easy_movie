Правила сайта

1. Общие положения

Регистрируясь на сайте, вы соглашаетесь с настоящими правилами. Администрация оставляет за собой право изменять правила без уведомления пользователей.

2. Выполнение заданий

  • Задание считается выполненным после подтверждения администратором
  • Запрещено использовать чужие данные для выполнения заданий
  • Одно задание можно выполнить только один раз
  • Попытка обмана системы ведёт к блокировке аккаунта без возврата средств

3. Вывод средств

  • Минимальная сумма вывода — 500 мандаринов
  • Вывод возможен только на карты, выпущенные в РФ
  • Срок обработки заявки — до 3 рабочих дней
  • В случае нарушения правил вывод может быть заблокирован

4. Ответственность

Администрация не несёт ответственности за действия партнёров, чьи задания размещены на сайте. Все претензии по качеству услуг решаются через поддержку.

⚠️ Блокировка вывода

Вывод средств может быть заблокирован в следующих случаях:

  • Использование нескольких аккаунтов одним лицом
  • Невыполнение условий задания
  • Попытка мошенничества
  • Нарушение правил партнёров
